# linear-regression-ml
# Linear Regression Project

## Objective

To implement and understand linear regression using machine learning.

## Tools Used

* Python
* Pandas
* Scikit-learn
* Matplotlib

## Dataset

Titanic Dataset

## Steps Performed

1. Data preprocessing and cleaning
2. Splitting data into training and testing sets
3. Training Linear Regression model
4. Evaluating model using MAE, MSE, and R² score
5. Visualizing regression results

## Results

* Model trained successfully
* Achieved reasonable prediction accuracy
* Observed relationships between features and target variable

## Conclusion

Linear regression is useful for predicting continuous values and understanding feature importance in datasets.
